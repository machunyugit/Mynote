## 环境

1. 阿里云服务器
2. linux-Centos。

## 移除原有git

```shell
git --version  	#查看当前版本号
yum remove git  #删除原有版本
```

## 下载git源码并解压

```shell
#切换 root 用户
su root
#下载git版本
wget -c https://github.com/git/git/archive/v2.25.1.tar.gz
#解压
tar zxf v2.25.1.tar.gz
```

## 编译安装

```shell
#进入git目录
cd git-2.25.1/
#配置git安装路径
./configure prefix=/usr/local/git/
#编译并且安装
make && make install
```

## 将git加入环境变量

```shell
#打开文件
vim /etc/profile
export PATH=$PATH:/usr/local/git/bin
#退出并执行
    source /etc/profile 
    #或者
    . /etc/profile
```

## 配置自动补全

```shell
#1.复制 git-completion.bash 文件到家目录
cp git-2.25.1/contrib/completion/git-completion.bash ～/.git-completion.bash
#2.打开家目录下的环境文件
vim ~/.bashrc
	#添加
	if [ -f ~/.git-completion.bash ]; then
    . ~/.git-completion.bash
    fi
 #3.更新
 . ~/.bashrc
```
