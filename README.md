## 山坡羊·潼关怀古（张养浩）

峰峦如聚，波涛如怒，山河表里潼关路。
望西都，意踌躇。
伤心秦汉经行处，宫阙万间都做了土。
兴，百姓苦；亡，百姓苦。

## 摸鱼儿（元好问）

问世间，情是何物，直教生死相许。
天南地北双飞客，老翅几回寒暑。
欢乐趣，离别苦，就中更有痴儿女。
君应有语，渺万里层云，千山暮雪。
只影向谁去。
横汾路，寂寞当年箫鼓，荒烟依旧平楚。
招魂楚些何嗟及，山鬼暗啼风雨。
天也妒，未信与，莺儿燕子俱黄土。
千秋万古，为留待骚人，狂歌痛饮。
来访雁丘处。

## 人生·江湖（黄沾）

天下风云出我辈，
一入江湖岁月催；
皇图霸业谈笑间，
不胜人生一场醉。
提剑跨骑挥鬼雨，
白骨如山鸟惊飞；
尘事如潮人如水，
只叹江湖几人回。
