#### 模板引擎加载的原理

1. 基于__autoload函数实现的文件的自动加载

   ```php
   //创建文件夹的名字为命名空间的
   function __autoload($className)
   {
       $className = str_replace('\\','/',$className);
       $file = './'.$className.'.php';
       if(file_exists($file)) {  
         include $file;
       }
   }

   $res = new \a\c\b\obj;
   $res = demo();
   ```

2. 基于 spl_autoload_register('autoload');

   ```php

    function autoload($className)
    {
    	$className = str_replace('\\','/',$className);
    	$file = './'.$className.'.php';

    	if (file_exists($file)) {
    		include $file;
    	} 
    }

    spl_autoload_register('autoload');
    
    $res = new \a\c\b\obj;
    $res -> demo();
   ```
