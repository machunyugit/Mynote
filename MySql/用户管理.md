## 架构概括

![丁奇](./image/丁奇.png)

1. Server 层包括连接器、查询缓存、分析器、优化器、执行器等，涵盖 MySQL 的大多数核心服务功能，以及所有的内置函数（如日期、数学和加密函数等），所有跨存储引擎的功能都在这一层实现，比如存储过程、触发器、视图。
2. 存储引擎层两部分。

### 连接器

连接器负责跟客户端建立连接、获取权限、维持和管理连接。

```mysql
mysql -h $ip -P $port -u $user -p
```

1. 查看连接状态 `show processlist`
